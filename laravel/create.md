# laravel新規プロジェクト作成

### プロジェクト作成  
```composer create-project laravel/laravel プロジェクト名 --prefer-dist "6.*"```  
この場合 version 6 の最新が入る(最後の部分でバージョン管理)  

### 起動  
```php artisan serve```

### jsを出力
```npm install```  
package.jsonに書かれたものがインストールされる  

### jsコンパイル
```npm run dev```  
走らせるとコンパイルされる  
```npm run watch```  
変更があると自動でコンパイルされる
